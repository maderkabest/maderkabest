# 💫 About Me:

👋 Hi, I’m Maria! I'm a **QA Automation Engineer** with a background in Cloud & DevOps, based in the Netherlands. 
I see myself as the "Guardian" of the codebase 🛡️✨. My goal is to build a solid perimeter of protection - through robust test automation and resilient CI/CD pipelines.

### 🔭 What I'm currently focused on:
* **Building Automation:** Developing stable E2E & API test frameworks using **Python**, **Playwright**, and **Pytest**.
* **Connecting QA & DevOps:** Integrating automated checks into CI/CD pipelines.
* **Continuous Learning:** Expanding my tech stack by mastering **TypeScript** for modern web automation.

⚡ *Fun fact: When I’m not writing tests or putting out infrastructure fires, you can usually find me at a yoga class or trying to master roller skating! 🛼*..
